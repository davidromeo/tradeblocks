# TradeBlocks MCP Server 2.2 — Market Data Provider Abstraction + Trade Replay + Greeks + Skills Plugin

Out of beta. Trade replay with minute-level greeks, exit trigger analysis across your whole portfolio, and a normalized VIX data model. Massive.com is the default data provider, but a provider abstraction layer lets you plug in alternatives (ThetaData, custom APIs). Everything works with locally cached bar data — no subscription required for replay and analysis.

**Market Data Provider Abstraction** — `import_from_api` replaces `import_from_massive`, backed by a pluggable `MarketDataProvider` interface. Set `MARKET_DATA_PROVIDER` env var to switch providers. CSV import still works as before.

**Trade Replay** — `replay_trade` resolves tradelog legs to OCC tickers and builds minute-level P&L paths with MFE/MAE. Cache-first — bars persist locally after first fetch.

**Greeks Engine** — per-leg delta, gamma, theta, vega, IV on every replay point. BS for regular options, Bachelier for 0DTE.

**Exit Trigger Analysis** — 14 trigger types (profit targets, stops, trailing stops, DTE/DIT, clock time, delta, VIX, S/L ratio). `batch_exit_analysis` tests policies across entire blocks with full aggregate stats. `decompose_greeks` breaks P&L into greek factor contributions with per-leg-group vega attribution for calendars.

**Profit Action Triggers** — partial closes via `closeAllocationPct`, directional delta exits via `exitAbove`/`exitBelow`.

**Bid/Ask Mid Mark Pricing** — replay uses bid/ask midpoint when available, HL2 fallback. Silent degradation if quote tier isn't available.

**Normalized VIX** — VIX tenors as regular ticker rows with `ivr`/`ivp`. Add VIX1D or VIX6M and enrichment discovers them automatically.

**Live Options Snapshot** — `get_option_snapshot` for real-time greeks/IV/OI with BS fallback.

**Other** — SQL confirm gate for mutations, ChatGPT compatibility, 60+ tools, 818 tests. One breaking change: `import_from_massive` renamed to `import_from_api`.

## Introducing TradeBlocks Skills Plugin (NEW)

[gitlab.com/davidromeo/tradeblocks-skills](https://gitlab.com/davidromeo/tradeblocks-skills)

Agent skills that chain MCP tools into guided analysis workflows. Works with Claude Code, Claude.ai, and other Agent Skills-compatible tools.

- **dc-analysis** — DC health check: exit attribution, S/L ratio, VIX regime fit, edge decay
- **health-check** — strategy health check: core metrics, Monte Carlo, position sizing
- **wfa** — walk-forward analysis: parameter robustness across rolling windows
- **optimize** — parameter exploration: sweep time, DTE, delta, VIX for patterns
- **portfolio** — portfolio analysis: correlation, diversification, marginal contribution
- **risk** — risk assessment: tail dependence, drawdown attribution, stress scenarios
- **compare** — strategy comparison: side-by-side metrics across blocks
- **market-data** — market data setup: import from API, CSV, or DuckDB
- **profile** — create strategy profiles from screenshots, descriptions, or block data

Install via Claude Code:
```
/plugin marketplace add davidromeo/tradeblocks-skills
/plugin install tradeblocks@tradeblocks-skills
```
