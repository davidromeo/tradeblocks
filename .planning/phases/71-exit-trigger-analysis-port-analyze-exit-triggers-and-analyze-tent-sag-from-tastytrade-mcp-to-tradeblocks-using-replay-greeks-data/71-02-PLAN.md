    **Core function: `decomposeGreeks(config: GreeksDecompositionConfig): GreeksDecompositionResult`**

    Implementation per D-09:

    1. Iterate consecutive pairs `(pnlPath[i], pnlPath[i+1])`:

       a. `actualChange = path[i+1].strategyPnl - path[i].strategyPnl`

       b. `underlyingChange`: Look up underlying prices for both timestamps. If not found, use 0 (delta/gamma will be 0 for this step).

       c. `dt`: Compute time difference in days between timestamps. For minute bars on same day, `dt = 1/390` (trading minutes per day). For cross-day gaps, compute actual calendar day fraction.

       d. `ivChange`: Average IV change across legs. For each leg, compute `legGreeks[i+1].iv - legGreeks[i].iv`. Average non-null values. Multiply by 100 to convert from decimal to percentage points (since vega is per 1% IV move).

       e. Per D-09:
          - `deltaPnl = (path[i].netDelta ?? 0) * underlyingChange`
          - `gammaPnl = 0.5 * (path[i].netGamma ?? 0) * underlyingChange * underlyingChange`
          - `thetaPnl = (path[i].netTheta ?? 0) * dt`
          - `vegaPnl = (path[i].netVega ?? 0) * ivChange`
          - `residual = actualChange - deltaPnl - gammaPnl - thetaPnl - vegaPnl`

    2. Accumulate step contributions into `FactorContribution[]`.

    3. **Per-leg-group vega per D-08:**
       If `legGroups` provided, for each group and each step:
       - Compute group vega: sum of `legGreeks[j].vega * legs[j].quantity * legs[j].multiplier / 100` for j in group's legIndices
       - Compute group IV change: average of `legGreeks[j+1].iv - legGreeks[j].iv` for j in group's legIndices (converted to % points)
       - Group vega P&L = groupVega * groupIvChange

    4. Sort factors by `abs(totalPnl)` descending per D-07.

    5. Compute `pctOfTotal` for each factor: `abs(totalPnl) / sum(abs(allTotalPnls)) * 100`.

    6. Build summary string: "P&L of ${totalPnlChange.toFixed(2)}: ${factor1.factor} ${factor1.totalPnl.toFixed(2)} (${factor1.pctOfTotal.toFixed(0)}%), ..."
