    **Create `packages/mcp-server/src/utils/black-scholes.ts`:**

    1. Implement the cumulative normal distribution function `cdf(x)` using the rational approximation (Abramowitz and Stegun 26.2.17) or the error function approach. This is the only non-trivial math dependency — no external library.

    2. Implement `bsPrice(type: "call" | "put", S: number, K: number, T: number, r: number, q: number, sigma: number): number`
       - S = underlying price, K = strike, T = time to expiry in years
       - r = risk-free rate (per D-07: default 0.045), q = dividend yield (per D-08: 0.015 for SPX, 0 otherwise)
       - Standard European BS formula with continuous dividend yield:
         - d1 = (ln(S/K) + (r - q + sigma^2/2) * T) / (sigma * sqrt(T))
         - d2 = d1 - sigma * sqrt(T)
         - call = S * e^(-qT) * N(d1) - K * e^(-rT) * N(d2)
         - put = K * e^(-rT) * N(-d2) - S * e^(-qT) * N(-d1)
       - Edge case: T <= 0 returns intrinsic value; sigma <= 0 returns intrinsic value

    3. Implement greeks functions — all take the same (S, K, T, r, q, sigma) signature:
       - `bsDelta(type, S, K, T, r, q, sigma): number` — N(d1)*e^(-qT) for call, (N(d1)-1)*e^(-qT) for put
       - `bsGamma(S, K, T, r, q, sigma): number` — N'(d1)*e^(-qT) / (S*sigma*sqrt(T))
       - `bsTheta(type, S, K, T, r, q, sigma): number` — standard BS theta formula, return per-day (divide annual by 365)
       - `bsVega(S, K, T, r, q, sigma): number` — S*e^(-qT)*N'(d1)*sqrt(T), return per 1% move (divide by 100)

    4. Implement `solveIV(type: "call" | "put", marketPrice: number, S: number, K: number, T: number, r: number, q: number, maxIter = 100, tolerance = 1e-6): number | null`
       - Newton-Raphson method (per D-11): sigma_new = sigma_old - (bsPrice - marketPrice) / vega
       - Initial guess: sigma = 0.3 (reasonable starting point for most options)
       - Guard: if vega < 1e-10, switch to bisection step to avoid division by near-zero
       - Return null if: marketPrice <= 0, T <= 0, no convergence after maxIter iterations
       - Convergence criterion: |bsPrice(sigma) - marketPrice| < tolerance

    5. Implement `computeLegGreeks(optionPrice: number, underlyingPrice: number, strike: number, dte: number, type: "C" | "P", riskFreeRate: number, dividendYield: number): GreeksResult`
       - Convert dte (fractional days, per D-09) to years: T = dte / 365
       - First solve IV from optionPrice
       - If IV is null, return { delta: null, gamma: null, theta: null, vega: null, iv: null }
       - Otherwise compute all greeks using solved IV and return them

    6. Export the `GreeksResult` interface:
       ```typescript
       export interface GreeksResult {
         delta: number | null;
         gamma: number | null;
         theta: number | null;  // per day
         vega: number | null;   // per 1% IV move
         iv: number | null;     // annualized implied volatility (0-N, not percentage)
       }
       ```
