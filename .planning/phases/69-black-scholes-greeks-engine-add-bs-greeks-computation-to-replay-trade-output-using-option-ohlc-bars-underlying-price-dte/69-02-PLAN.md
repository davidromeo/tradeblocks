            - DTE = (expiryDate - barDate) / (1000 * 60 * 60 * 24)
            - If DTE <= 0, set greeks to null result
          - Call `computeLegGreeks(legPrices[i], underlyingPrice, greeksConfig.legs[i].strike, dte, greeksConfig.legs[i].type, greeksConfig.riskFreeRate, greeksConfig.dividendYield)`
       c. Compute net position greeks (per D-02, using quantity * multiplier / 100 per the specifics note):
          ```typescript
          let netDelta = 0, netGamma = 0, netTheta = 0, netVega = 0;
          let allNull = true;
          for (let i = 0; i < legGreeksArr.length; i++) {
            const g = legGreeksArr[i];
            if (g.delta !== null) {
              allNull = false;
              const weight = legs[i].quantity * legs[i].multiplier / 100;
              netDelta += g.delta * weight;
              netGamma += g.gamma! * weight;
              netTheta += g.theta! * weight;
              netVega += g.vega! * weight;
            }
          }
          ```
       d. Get IVP for this date: `greeksConfig.ivpByDate?.get(dateStr) ?? null`
       e. Set on the PnlPoint: `legGreeks`, `netDelta`, `netGamma`, `netTheta`, `netVega`, `ivp`
